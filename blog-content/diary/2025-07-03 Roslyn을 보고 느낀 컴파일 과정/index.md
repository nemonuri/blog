## 토큰화

- 입력
  - 문자열
  - 토큰화 규칙
- 반환
  - 토큰 나열
- 설명
  - 일종의 메모라이징 기법
  - 구문 분석의 성능 향상을 위해, 문자열을 날것 그대로 사용하는 대신, 문자열을 분할해 '토큰 나열'을 만들고, 그 토큰 나열을 구문 분석의 입력으로 사용한다.
  - 즉, '토큰'이 새 '문자'다.

## 구문 분석

- 입력
  - 토큰 나열
  - 구문 분석 규칙
- 반환
  - 구문 트리
- 설명
  - 구문 분석에 대한 이론은 찾아보면 많다. 넘어가자.

## 바인더 포레스트 제작

- 입력
  - 구문 트리
  - 바인더 팩토리
  - 바인더 생성 규칙
- 반환
  - 바인더 포레스트
- 설명
  - '바인더'는 다음 정보를 가진다.
    - 자신과 연결된 구문 노드
    - 자신의 부모 바인더
  - '용도'에 따라, 한 구문 노드가 여러 바인더와 연결될 수 있다.
  - 바인더 포레스트는, (구문 노드, 용도)를 Key로, 바인더를 Value로 하는 '캐시 맵' 이다.
